# **Reading Notes for 102d45**

*In this lesson we are learning how to use Markdown to make a website.*
  

## **Below are some common questions and markdown solutions.**

### *How to make a header?*

```markdown
<!-- markdown -->
# header1
## header2
### header3
#### header4
```
`<!-- rendered -->`
# header1
## header2
### header3
#### header4


### *How to make a table?*

```markdown
<--! markdown -->
|column_name1    |column_name2   |column_name3  |
|:---------------|--------------:|:------------:|
|right-justified | left-justified|  centered    |
```
`<!-- rendered-->`
|column_name1    |column_name2   |column_name3  |
|:---------------|--------------:|:------------:|
|right-justified | left-justified|  centered    |


### *How to make a linked image?*

```markdown
<!-- markdown -->
<!-- to make a link -->
[object](address)

<!-- to make an image -->
![text](address)

<!-- combined them -->
[![alternate text](
```
